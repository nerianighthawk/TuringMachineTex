\begin{defi}[Turing Machineの定義]
$M:=(Q,\Gamma,b,\Sigma,\delta,q_{0},F)$が以下を満たす時, $M$をTuring Machine という.
\begin{enumerate}
\item $Q$は有限かつ空でない集合とする.
$Q$の元を状態と呼ぶ.
\item $\Gamma$は有限かつ空でない集合とする.
$\Gamma$の元を記号と呼ぶ.
\item $b\in \Gamma$が存在する.
$b$を空白記号と呼ぶ.
\item 
\end{enumerate}
\end{defi}
